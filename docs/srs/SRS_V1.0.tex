% --- SECTION 1: INTRODUCTION ---
\section{Introduction}

\subsection{Purpose}
The purpose of this Software Requirements Specification (SRS) is to outline the functional and external interface requirements for the Lunar Rocks! system. It defines the scope, expected behavior, and constraints of the software for Snapshot 1 development.

\subsection{Intended Audience}
This document is intended for:
\begin{itemize}
    \item Software developers working on frontend, backend, or database components.
    \item Project managers overseeing Snapshot 1 progress and requirements.
    \item QA testers evaluating whether the system meets specifications.
\end{itemize}

\subsection{Overview of the Software}
Lunar Rocks! is a web-based citizen science tool enabling users to analyze lunar images from NASA missions. The application allows users to register, log in, scout images, size rocks, classify rock shapes, and submit results to a centralized scientific dataset.

\subsection{Definitions, Acronyms, Abbreviations}
\begin{itemize}

    \item \textbf{API} — Application Programming Interface

    \item \textbf{NASA} — National Aeronautics and Space Administration

\subsection{Software Interfaces}
The system interacts with:
\begin{itemize}
    \item A Java backend service running on Tomcat.
    \item A PostgreSQL database containing user, image, and rock classification data.
    \item NASA-hosted image datasets (future expansion).
\end{itemize}

\subsection{Communications Interfaces}
\begin{itemize}
    \item HTTPS for secure client-server communication.
    \item Email contact for users needing support.
\end{itemize}

% --- SECTION 3: LEGAL AND ETHICAL CONSIDERATIONS ---
\section{Legal and Ethical Considerations}

\subsection{Data Privacy}
\begin{itemize}
    \item User accounts require only minimal personal information.
    \item Passwords must be securely hashed.
    \item No user-generated scientific data is linked to identifiable personal information.
\end{itemize}

\subsection{Accessibility}
The system follows WCAG and USWDS guidance:
\begin{itemize}
    \item Adequate color contrast.
    \item Keyboard navigability.
    \item Descriptive labels and alt-text for all images.
\end{itemize}

\subsection{Ethical Use}
\begin{itemize}
    \item Citizen science contributions must be used responsibly.
    \item Data must be preserved with integrity for scientific use.
\end{itemize}
